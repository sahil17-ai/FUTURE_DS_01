# Task 1: Business Sales Analysis (E-commerce)

## Objective
Analyze e-commerce sales data to identify sales trends,
best-selling products, and high-revenue countries.

## Dataset
Online Retail II Dataset

## Tools Used
- Python
- Pandas
- Seaborn
- Matplotlib
- Google Colab

## Steps Performed
- Data cleaning and preprocessing
- Feature creation (Sales)
- Exploratory Data Analysis
- Data visualization

## Key Insights
- Identified top countries by sales
- Observed monthly sales trends
- Found best-selling products

## Conclusion
This project demonstrates how data analysis can help
businesses make data-driven decisions.
